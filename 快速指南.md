# 电机调试工具 - 快速使用指南

> **给新手的话：** 本工具就像一个"电机遥控器"，可以帮你查看和设置电机。不需要懂编程，按照步骤操作即可。

---

## 🔌 第一步：连接电机

1. **用USB线连接**：把电机控制器用USB线连接到电脑
2. **给电机供电**：确保电机有电源供应（很重要！）
3. **找到串口号**：

   **Windows 用户：**
   - 右键点击"此电脑" → 管理 → 设备管理器
   - 展开"端口(COM和LPT)"
   - 记住端口号，比如 `COM3`、`COM4` 等
   
   **Linux 用户：**
   - 通常是 `/dev/ttyACM0` 或 `/dev/ttyACM1`
   - 不确定的话，在程序里试试看

---

## 🎮 第二步：使用工具

### 启动程序

- **Windows**：双击 `motor_tool.exe`
- **Linux**：双击 `motor_tool` 或在终端输入 `./motor_tool`

程序会显示一个菜单，像这样：

```
============================================================
           电机调试工具 v1.2
============================================================
1. 查看/修改串口配置
2. 扫描电机
3. 设置电机ID和Master ID
4. 设置单个电机零点
5. 设置全部已连接电机零点
0. 退出
============================================================
请选择功能 (0-5):
```

### 第一次使用：配置串口

**智能检测**：程序启动时会自动检测串口连接状态
- ✅ 如果串口已连接：直接进入主菜单
- ⚠️ 如果串口未连接：自动进入配置菜单，提示你设置正确的串口

**手动配置步骤**：
1. 在主菜单输入 `1`（查看/修改串口配置），按回车
2. 输入 `1`（修改串口端口）
3. 输入你的串口号：
   - Windows 用户输入：`COM3`（或你在设备管理器看到的号）
   - Linux 用户输入：`/dev/ttyACM1`（或其他）
4. 也可以输入 `4`（列出所有可用串口）查看并选择
5. 输入 `0` 返回主菜单

> 💡 **只需要配置一次**，程序会记住你的设置

## 📖 常用功能说明

### 功能2：扫描电机

**用途**：查看总线上有哪些电机

**步骤**：
1. 在主菜单输入 `2`，按回车
2. 等待几秒，会显示找到的电机列表

**结果示例**：
```
找到 1 个电机

电机列表:
  Motor ID | Master ID |  当前位置    | 状态
 --------------------------------------------------
   0x01    |  0x11     |  -8.334 rad | 正常
```

---

### 功能3：设置电机ID（重要）

**用途**：给电机改名字（修改ID号和Master ID）

**重要提示**：
- ⚠️ 这个操作会**永久修改**电机的ID（保存到Flash）
- ✅ 修改后即使断电重启，新ID也不会丢失
- Master ID 自动计算：`Master ID = 0x10 + Motor ID`（十六进制）
  - 例如：Motor ID 为 0x1，则 Master ID 自动设为 0x11

**步骤**：
1. 主菜单输入 `3`
2. 输入**当前电机ID**（比如 `1`）
3. 输入**新的电机ID**（支持十进制或0x十六进制，比如 `5` 或 `0x05`）
4. 确认修改：系统会显示新旧ID对比，输入 `y` 确认
---

### 功能4：设置单个电机零点

**用途**：把指定电机的当前位置设为起始点（0度位置）

**步骤**：
1. **先手动转动电机**到你想要的零点位置
2. 主菜单输入 `4`
3. 输入电机ID（比如 `1`）
4. 自动设置，无需确认
5. 看到 `[OK] 零点设置成功！` 即完成

---

### 功能5：设置全部已连接电机零点（批量操作）

**用途**：一次性为所有已连接的电机设置零点

**特点**：
- 🔍 自动扫描所有已连接的电机
- 📋 显示检测到的电机列表和当前位置
- ✅ 需要确认后才执行

**步骤**：
1. **先手动转动所有电机**到各自想要的零点位置
2. 主菜单输入 `5`
3. 程序自动扫描所有已连接电机
4. 显示检测到的电机列表，例如：`检测到 3 个电机: [1, 2, 3]`
5. 提示确认：输入 `y` 确认，或直接回车（默认确认）
6. 程序显示设置前后的位置对比表，设置完成

> 💡 **适用场景**：需要同时校准多个电机时非常方便，省时高效

## ❓ 常见问题

### 问题1：程序打不开
**解决办法**：
- Windows：右键点击程序，选择"以管理员身份运行"
- Linux：确保文件有执行权限，在终端运行 `chmod +x motor_tool`

### 问题2：找不到串口
**解决办法**：
1. 检查USB线是否插好
2. 确认设备管理器能看到设备
3. 尝试换个USB口

### 问题3：扫描不到电机
**检查清单**：
- ✅ 电机是否通电（很重要！）
- ✅ USB线是否连接好
- ✅ 串口号是否正确
- ✅ 波特率是否正确（默认921600）

### 问题4：Linux提示没权限
**解决办法**：
在终端输入以下命令，然后重新登录：
```bash
sudo usermod -a -G dialout $USER
```

## 💡 小贴士

1. **先扫描，后操作**：不确定电机ID时，先用功能2扫描一遍
2. **智能启动检测**：程序启动时会自动检查串口连接，未连接会直接进入配置菜单
3. **改ID要小心**：修改ID前确认当前ID，避免改错
5. **批量设零点**：使用功能5可以一次性为所有电机设置零点，省时高效
5. **批量设零点**：使用功能6可以一次性为所有电机设置零点，省时高效
6. **可以重复使用**：同一个程序可以复制到其他电脑使用
7. **断电重启没影响**：设置的ID和零点会永久保存，不用担心断电丢失
